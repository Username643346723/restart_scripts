Инструкция по настройке и запуску скриптов

1. Настройка виртуального окружения

    Запустите файл setup_env.bat двойным кликом.
    Откроется командная строка. Скрипт:
        Создаст виртуальное окружение;
        Установит все необходимые библиотеки из файла requirements.txt.
    Дождитесь завершения работы скрипта (появится сообщение о завершении). Нажмите любую клавишу, чтобы закрыть окно.
2. Запуск скриптов

    Дважды кликните на файл перезапуск_скриптов.bat.
    Откроется командная строка с меню:
    
    Выберите скрипт для запуска:
    [1] перезапустить_гороскоп.py
    [2] перезапустить_копер_постов_с_тг_в_вк.py
    [3] перезапустить_копер_постов_с_тг_группы_ejdailyru.py
    [4] перезапустить_подписка_бот.py
    Введите номер скрипта (например, 1) и нажмите Enter.
    Скрипт запустится. Дождитесь завершения его работы.
    После выполнения нажмите любую клавишу, чтобы закрыть окно.

Если ничего не происходит при запуске:
